Conference Venue

Location

National Institute of Informatics
2-1-2 Hitotsubashi, Chiyoda-ku, Tokyo 101-8430
☎ +81-3-4212-2000

Airport Access

Access from Haneda Airport: The conference venue is reachable using the Keikyu-Kuko and Mita lines (change at Mita station).

Access from Narita Intl Airport: Tokyo Station is conveniently reachable using the Narita Express from either airport terminal. The conference venue is reachable from Tokyo Station via the Tokyo Metro or a short taxi ride. Please note that taxi rides directly from Narita Airport will be prohibitively expensive.

Metro Access

An english map of the Tokyo Metro network is available here. The conference venue is located within walking distance from the following Metro stations.
